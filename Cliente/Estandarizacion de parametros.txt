Este archivo puede tener actualizaciones segun se vea necesario. ¡LEEALO POR FAVOR!

--Los archivos css se colocaran en la siguiente ruta:
	dist/css/
	(si es algun plugin con varios archivos css cree su respectiva subcarpeta)



--Los archivos js se colocaran en la siguiente ruta:
	dist/js/
	(si es algun plugin con varios archivos css cree su respectiva subcarpeta)



--Las imagenes se guardaran en la siguiente ruta:
	- Para los usuarios assets/images/users 
	- Para fondos assets/images/background
	- Los logos puede colocarlos dentro assets/images/
	- Para los autos assets/images/autos/ En su respectiva carperta con el nombre compuesto por la marca modelo y ano ejemplo:
			assets/imagenes/autos/Mazda CX-3/
